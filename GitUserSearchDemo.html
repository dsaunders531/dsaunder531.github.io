GitUserSearchDemo

This project was created as part of a recuitment process.

Requirements

Create an ASP.Net MVC website with a page containing a text box to enter a name in and a submit button to search GitHub for the name.

Have the back end call the GitHub users API (e.g. https://api.github.com/users/facebook) and get the users name, location and avatar url from the returned json. Use the repos_url value to get a list of all the repos for the user.

On the results page, show the username, location, avatar and the 5 repos with the highest stargazer_count.

Use this opportunity to show us what you know, even if you wouldn’t ordinarily use the concepts in such a trivial example.

Solution

An ASP.Net Core MVC site, the front-end uses bootstrap. Views use Razor syntax.

After reading the Github api documentation which is comprehensive, I thought that it would take a long time to make a http client to handle the requests. Luckily, there is a library aleady which can be used to do the processing work. Octokit saved me a huge amount of time and made it much easier.

Use

The site is very simple. Users login using thier GitHub user name and password. These details are not stored in the app.

After logging in users are presented with a search box to find a user. If one is sucessfully found, a results page is shown.
